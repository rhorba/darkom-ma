# Darkom.ma — Property Rental Management SaaS

Morocco's rental market is 90% paper-based. Landlords manage multiple properties through WhatsApp messages.

## Problem
No affordable tool for Moroccan landlords to track leases, collect rent, manage maintenance requests, or generate compliant lease contracts.

## Solution
SaaS dashboard for landlords, tenant self-service portal, automated rent reminders, CMI online rent payment, and lease document generation under Moroccan tenancy law.

## Stack
Angular 19 (standalone components, Angular Material), Java 21 + Spring Boot 3, PostgreSQL 16 + Flyway, Spring Security (JWT), CMI payments, Docker Compose (localhost) — Kubernetes deferred until actually needed

## Completes
Bina (construction → Darkom manages the resulting rentals)

## Key Roles
Landlord | Tenant | Property Manager | Admin
